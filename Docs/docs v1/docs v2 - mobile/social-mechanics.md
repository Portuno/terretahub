# Social Mechanics (co-op y competencia sana)

## Objetivo
Usar lo social para aumentar motivación y pertenencia, no para humillar o forzar grind.

## 1) Amistades y equipos
- Permitir “misión con un amigo” (1 check compartido o una meta conjunta).
- Equipos por temporada (equipo A/B) para cooperación y narrativas simples.

## 2) Desafíos 1v1 con contexto
Ejemplos:
- “Completar 3 misiones diarias antes de que termine el día”.
- “Sumar XP de exploración en 24 horas”.

Reglas anti-frustración:
- El rival es “fair”: la competición se ajusta a nivel o historial aproximado.
- Mostrar el progreso en tiempo real (o casi) para que el usuario entienda el estado.

## 3) Leaderboards con ventana temporal
- Ranking por día/semana/temporada.
- Evitar “ranking total infinito” en mobile (se siente imposible de alcanzar y genera abandono).

## 4) Co-op que da valor incluso si el otro no está
- Si un amigo no completa, el equipo no “pierde todo”: se asigna progreso parcial.
- Recompensas escalonadas: por participación y por objetivo completo.

## 5) Compartir logros como identidad
- Mostrar logros de perfil como badges visibles.
- Facilitar “compartir” (captura o link) con texto automático claro (sin clickbait).

## 6) Moderación y cultura
- Mensajes de “encendido” deben ser positivos: celebrar progreso, no castigar fallas.
- Evitar notificaciones sociales ofensivas o spam.
