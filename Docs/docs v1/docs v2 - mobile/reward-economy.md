# Reward Economy (monedas, recompensas y fairness)

## Principio rector
La economía debe sostener hábito sin castigar al jugador que no paga.

## Modelo recomendado (simple y balanceado)
### Moneda blanda (progresión diaria)
Se gana con:
- misiones diarias,
- misiones semanales,
- logros,
- participación social (sin forzar).

Uso:
- desbloqueos cosméticos,
- cofres/badges,
- boosters de bajo impacto (opcional y limitado).

### Moneda dura (si aplica)
Usar solo si el producto necesita una monetización.
Reglas recomendadas:
- si existe, que el valor se centre en cosméticos o conveniencia leve,
- evitar que compre “progreso de poder” de forma directa.

## Tipos de recompensas (qué conviene para empezar)
- **Cosméticos** (recomendado): badges, marcos, temas, estilos de perfil, animaciones sutiles.
- **Colecciones**: conjuntos visuales (ej. “colección de verano”).
- **Accesos**: recompensas que habilitan herramientas pequeñas (no punitivas).

## Recompensas que se sienten bien (diseño de entrega)
- Mostrar el cambio visual o el desbloqueo inmediatamente.
- Explicar la “razón” (por qué la persona la ganó).
- Evitar “hoarding” sin significado: si hay cofre, que tenga odds claros o que sea determinista.

## Guardrails anti-pay-to-win
- Si hay monetización, que no afecte misiones diarias de manera injusta.
- Evitar multiplicadores que rompan el pacing (por ejemplo XP multiplicado sin límites).

## Economía como narrativa
La moneda no es el fin: es el vehículo.
Cada temporada debería tener:
- una historia ligera,
- una colección principal,
- recompensas escalonadas por participación.
